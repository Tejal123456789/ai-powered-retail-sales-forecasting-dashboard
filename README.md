# AI-Powered Retail Sales Analytics & Forecasting Dashboard

## Project Overview
This project is an end-to-end retail sales analytics and forecasting solution built using MySQL, Python, Machine Learning, and Power BI. The project analyzes historical retail sales data, cleans and transforms it using SQL, performs exploratory data analysis in Python, applies machine learning models for sales prediction, and visualizes business insights and forecasts through an interactive Power BI dashboard.

## Objectives
- Clean and preprocess raw retail sales data using MySQL
- Perform business insight analysis using SQL and Python
- Build machine learning models to predict sales trends
- Create an interactive Power BI dashboard for descriptive and predictive analytics
- Generate future sales forecasts for business decision-making

- `retail_sales.csv` → original raw dataset

### 3. Exploratory Data Analysis
